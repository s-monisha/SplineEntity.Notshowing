libdxfrw Documentation

This manual documents the use of libdxfrw.

With libdxfrw you can read and write several parts of a dxf files.

Dxf files can be written in assci and binary form, both are supported.

Dwg support (only read) are work in progress.

the complete documentation and examples are pending to free time, but to start see DRW_Interface, dxfRW & dwgR, clases
